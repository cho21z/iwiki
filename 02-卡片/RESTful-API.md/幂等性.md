> 💡 什么是幂等？为什么同一个请求重复发多次，结果应该保持一致？哪些 HTTP 方法是幂等的？

## 幂等性（Idempotent）
同一个请求，无论执行多少次，结果都是一样的。

## Method 幂等性分类
| Method | 是否幂等 | 说明 |
| --- | --- | --- |
| GET | 是 | 查询不应该改变资源状态 |
| PUT | 是 | 相同请求重复执行结果一致 |
| DELETE | 是 | 删除多次，资源最终都不存在 |
| POST | 通常否 | 可能会重复创建资源 |
| PATCH | 不一定 | 取决于补丁语义 |

## 为什么要关注幂等性？
因为在真实生产环境中，请求可能会因为网络问题、服务器故障等原因失败，或者用户重复点击提交按钮、消息系统重复消费，如果没有幂等设计，就会出现重复下单、重复扣款、重复发消息等问题。
