# 计算机网络

> 后端开发的"通用语"——任何分布式问题的根，最终都会落到网络上。

对应学习阶段：**阶段 1** —— 计算机基础（6-8 周）

---

## 📍 学习路径

```
阶段一：分层模型与基本概念（建立心智地图）
   ├─ OSI / TCP-IP 分层
   ├─ 一次 HTTP 请求的完整链路
   └─ 各层协议的职责边界

阶段二：传输层（TCP/UDP，重点中的重点）
   ├─ TCP 三次握手 / 四次挥手
   ├─ 可靠传输机制（滑动窗口、拥塞控制、重传）
   ├─ TCP 状态机
   └─ UDP 与适用场景

阶段三：应用层（HTTP 全家桶）
   ├─ HTTP 1.0 / 1.1 / 2 / 3 演进
   ├─ HTTPS 与 TLS 握手
   ├─ DNS 解析全过程
   └─ WebSocket 与长连接

阶段四：实战与排查
   ├─ 抓包工具（tcpdump / Wireshark）
   ├─ 常见网络故障定位
   └─ 性能调优（Keep-Alive、连接复用、TCP 参数）
```
