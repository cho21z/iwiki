# 操作系统

> 理解程序运行的物理世界——进程怎么跑、内存怎么管、I/O 怎么走、Linux 怎么用。

对应学习阶段：**阶段 1** —— 计算机基础（6-8 周）

---

## 📍 学习路径

```
阶段一：进程与线程（理解"程序在跑什么"）
   ├─ 进程 vs 线程
   ├─ 进程调度
   ├─ 上下文切换
   └─ 进程间通信
            ↓
阶段二：内存管理（理解"程序在哪里跑"）
   ├─ 虚拟内存与物理内存
   ├─ 分页机制
   ├─ 缺页与换入换出
   └─ 内存映射 mmap
            ↓
阶段三：I/O 与文件系统（理解"程序怎么和外部交互"）
   ├─ 阻塞 / 非阻塞 / 多路复用
   ├─ select / poll / epoll
   ├─ 零拷贝
   └─ 文件系统基础
            ↓
阶段四：并发与同步（理解"多个程序怎么协作"）
   ├─ 锁、信号量、条件变量
   ├─ 死锁的产生与避免
   └─ 用户态 vs 内核态
            ↓
阶段五：Linux 实战（落地到日常工作）
   ├─ 常用命令与组合
   ├─ 文件权限与用户
   ├─ Shell 脚本
   └─ 性能排查命令（top、iostat、vmstat、strace）
```
